# Curriculum Text Copyright Review Plan

This pass is for content safety, not technique accuracy alone.

## Current Ground Truth

- Curriculum text was written by the developer and expanded with ChatGPT assistance.
- No app logo from a known third-party source should remain in active use.
- No technique photos or third-party instructional screenshots are intentionally included in the app.
- Screenshots and internal docs were created during product work on this app.

## Main Risk Areas

1. Verbatim third-party teaching copy
Examples:
- paid instructional summaries
- book excerpts
- gym curriculum text pasted into entries
- YouTube transcript wording

2. Coach-branded/system-branded description voice
Examples:
- entries that sound like course-page language
- “system” descriptions that read like marketing copy instead of app coaching notes

3. Long AI-generated descriptions that are too polished or generic
Examples:
- text that does not sound like the rest of the app
- text that explains too much in a lecture/sales tone instead of a coaching-note tone

## Rewrite Standard

Each entry should read like original coaching notes for this app:

- what the position or technique is for
- what the learner should look for
- what it connects to next
- what the major risks or reactions are

Avoid:

- course-page tone
- source-dependent phrasing
- long lecture-style exposition
- wording that sounds copied from a named instructor’s framework

## Review Checklist

- [ ] No copied/verbatim third-party material remains
- [ ] No paid instructional copy is reproduced in descriptions
- [ ] No YouTube/book/article transcript text is pasted in
- [ ] Named systems are described in app-specific language
- [ ] Longer entries sound like concise coaching notes
- [ ] Descriptions stay consistent with the rest of the app voice

## Already Started

The first cleanup pass has already begun in the highest-risk cluster:

- named modern guards
- named backside routes
- back-control system descriptions
- tren-lock wording

## QA Goal

After each rewrite pass, the question should be:

“Does this sound like our own gym/app coaching note?”

If yes, it is usually in a much safer place.
